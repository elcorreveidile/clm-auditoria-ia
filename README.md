## 📋 Descripción

Formulario interactivo de auditoría sobre el uso de Inteligencia Artificial en el Centro de Lenguas Modernas (CLM) de la Universidad de Granada. Diseñado con 9 caminos específicos para recopilar información relevante según el área de trabajo de cada persona.

---

## 🎯 Características

- **9 Caminos Específicos:** Cada área tiene preguntas adaptadas a su realidad
- **Relevancia 95-100%:** Preguntas específicas, sin "no aplica"
- **192 Preguntas Totales:** Distribuidas en 33 secciones
- **Responsive:** Funciona en móvil, tablet y escritorio
- **Fácil Mantenimiento:** Estructura modular y documentada

## 🛠️ Instalación y Uso

### Desarrollo Local

```bash
# Clonar repositorio
git clone https://github.com/elcorreveidile/clm-auditoria-ia.git
cd clm-auditoria-ia

# Abrir en navegador
open index.html  # macOS
start index.html # Windows
xdg-open index.html # Linux
```

### Producción

**El formulario ya está en producción en:** https://clm.laclasedigital.com

Para actualizar:

```bash
# 1. Hacer cambios
vim index.html

# 2. Probar localmente
open index.html

# 3. Commit
git add index.html
git commit -m "descripción de cambios"

# 4. Push
git push origin main

# Los cambios se reflejan automáticamente en clm.laclasedigital.com
```
